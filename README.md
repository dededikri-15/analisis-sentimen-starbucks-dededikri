Analisis Sentimen Starbucks Indonesia
Proyek Python untuk analisis sentimen dan data mining berdasarkan ulasan aplikasi Starbucks Indonesia. Repository ini mencakup proses pengumpulan data, preprocessing, eksplorasi data (EDA), analisis sentimen, visualisasi, serta dokumentasi hasil analisis.


Tujuan Analisis
Proyek ini bertujuan untuk:
- Mengidentifikasi kecenderungan sentimen pengguna terhadap aplikasi Starbucks.
- Mengelompokkan pola keluhan atau masukan pengguna ke dalam klaster tertentu menggunakan algoritma K-Means.
- Menemukan hubungan antar kata kunci dalam ulasan yang sering muncul bersamaan menggunakan algoritma Apriori.
- Memberikan rekomendasi berbasis data untuk peningkatan kualitas layanan digital Starbucks di masa mendatang.
